PRD GAME RETRO PATH

1 INFO PRODUK
Nama: Retro Path
Genre: Puzzle Strategi
Tema Visual: Pixel Art Retro
Platform Target: Desktop EXE dan AppImage
Teknologi Utama: HTML CSS JavaScript dan Electron
Penyimpanan Data: Local Storage

2 DESKRIPSI GAME
Kamu membuat game teka teki penemuan jalur.
Karakter pemain berdiri di titik awal.
Pemain menempatkan blok jalan di atas area kosong.
Pemain menyambungkan titik awal menuju tujuan akhir.
Game ini menggunakan gaya visual piksel klasik.

3 MEKANIK UTAMA
Sistem menampilkan papan area bermain berukuran kotak kotak.
Sistem memberikan pilihan blok jalan acak kepada pemain.
Pemain memindahkan blok jalan ke atas papan.
Pemain menekan tombol mulai jalan.
Karakter bergerak maju mengikuti jalur buatan pemain.
Karakter mencapai tujuan akhir.
Sistem memberikan hadiah berupa koin permainan kepada pemain.

4 FITUR TOKO TIRUAN MARKETPLACE
Kamu membuat menu toko di dalam layar game.
Toko ini mensimulasikan proses pembelian menggunakan uang asli.
Sistem menulis dan membaca semua riwayat transaksi dari Local Storage.
Kamu menambahkan 4 kategori produk di dalam toko.
Kategori A adalah Mata Uang. Pemain membeli tumpukan koin atau permata.
Kategori B adalah Paket Level. Pemain membuka tahap teka teki tingkat lanjut.
Kategori C adalah Bantuan. Pemain membeli fitur petunjuk jalur yang benar.
Kategori D adalah Kostum. Pemain mengubah tampilan visual karakter utama.

5 INSTRUKSI PEMROGRAMAN UNTUK AGEN AI
Kamu menggunakan dokumen ini sebagai panduan utama pembuatan kode.
Kamu menyiapkan proyek Node JS kosong.
Kamu menginstal Electron sebagai dependensi proyek.
Kamu membuat file index html untuk menampung elemen kanvas dan antarmuka toko.
Kamu membuat file style css untuk mengatur tata letak dan memberikan efek retro.
Kamu membuat file game js untuk memproses logika pergerakan karakter dan sistem kisi.
Kamu membuat file shop js untuk mengelola objek barang dan memperbarui saldo Local Storage.
Kamu mengonfigurasi file main js untuk menjalankan jendela aplikasi Electron.
Kamu menulis skrip pengaturan electron builder di dalam package json.
Kamu memastikan skrip berhasil mengemas proyek menjadi file EXE untuk Windows dan AppImage untuk Linux.
